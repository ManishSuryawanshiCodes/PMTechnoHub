# PM TECHNO HUBB

Multi-page static site — IoT, robotics & STEM workshops, Pune.

**GitHub:** [baryongaming/PMTechnoHub](https://github.com/baryongaming/PMTechnoHub)  
**Maps:** [Google Maps listing](https://maps.app.goo.gl/XvjBEcS1RpoRpRNu6)

## Quick start

```bash
npx serve .
# open http://localhost:3000
```

## Edit content

| What | Where |
|------|--------|
| Services / workshops / products / gallery | `js/data.js` |
| Phone, email, maps URL | `js/config.js` |
| Styles | `css/`, `design.md` |

**Gallery (no DB):** add JPGs to `images/gallery/`, list paths in `data.js` → `galleryProjects`.
